# Manarem - Recomendaciones de Anime, Manga y Música

Plataforma web para descubrir y recomendar animes, mangas y música relacionada.

## Arquitectura

- **Frontend**: HTML5, CSS3, JavaScript vanilla - Desplegado en Vercel
- **Backend**: Python + Flask + PostgreSQL - API REST en VPS (en preparación)

## Tecnologías

- HTML5, CSS3, JavaScript vanilla
- Python 3 / Flask
- PostgreSQL
- Docker / Docker Compose
- Traefik (reverse proxy)
- Cloudflare (DNS/CDN)
